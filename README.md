# Media Interaktif Bangun Ruang

Media pembelajaran interaktif berbasis web untuk materi bangun ruang (segitiga, persegi dan persegi panjang), dilengkapi kuis dan game edukatif.

## Akses Project

👉 [https://media-interaktif-bangun-ruang.netlify.app/](https://media-interaktif-bangun-ruang.netlify.app/)

## Menu

### Materi
Penjelasan interaktif tentang bangun ruang.

### Kuis
- **Soal Cerita** : Soal berbasis cerita untuk menguji pemahaman konsep
- **Soal Duel** : Mode 2 pemain, 5 soal pilihan ganda adu cepat

### Game
- **Cocokkan Gambar** : Drag and drop gambar ke pasangan yang sesuai
- **Susun Puzzle** : Menyusun potongan gambar dengan cara drag and drop

## Dibuat dengan
- Articulate Storyline
- HTML, CSS, JavaScript
